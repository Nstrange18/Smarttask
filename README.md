SmartTask is a modern, clean, and intuitive task management web application designed to help users organize their daily activities, track progress, and visualize workflows.

### ðŸ—‚ Task Management

## Installation & Setup

### Clone the repository
```bash
git clone https://github.com/Nstrange18/smarttask.git
```
### Change Directory
```bash
cd smarttask
```

### Installation
```bash
npm install
```

### Run the development server
```bash
npm run dev
```
